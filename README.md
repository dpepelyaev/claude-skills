## Установка

Для любого скилла из каталога:

```bash
cd <папка-скилла>
bash install.sh
```

Скрипт установит:
- Необходимые зависимости (npm пакеты, конфиги)
- Регистрацию в Claude Code (`~/.claude/skills/`)
- Smoke-тесты работоспособности

После установки скилл **доступен как `/slash-command`** в Claude Code.

## Быстрый старт

```bash
# Проверь статус установки
pm2 list

# Перезагрузи Claude Code
/clear

# Используй скилл (пример для token-optimizer)
/token-optimizer "текст задачи"
```
